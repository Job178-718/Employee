# SSM-CURD
## 技术
+ 后端：
  >   java,SSM框架，json数据封装javaBean
+ 数据库：
  > mysql
+ 前端：
  > 1.HTML+CSS+JS
        
  > 2.Jquery/bootstrap前端框架
   
## 逻辑
+ 查询
  + 业务层：Service   
  + 持久层：Dao
  + View层：Controller
  > 启动服务器默认返回index.jsp,前端通过Ajax对页面进行响应，Ajax调用控制器，控制器通过业务层调用持久层进行查找,
    最终控制器将数据以JavaBean封装好的json形式进行返回。
